# Active-Directory-Lab
I recently set up a server backup using Windows Server Backup and wanted to share the process I followed.

1️⃣ Open Server Manager
2️⃣ Go to Manage > Add Roles and Features
3️⃣ Install Windows Server Backup
4️⃣ Open Windows Server Backup
5️⃣ Click Backup Schedule
6️⃣ Choose the backup configuration
7️⃣ Select the items you want to protect
8️⃣ Set the backup time
9️⃣ Choose the backup destination
🔟 Review the settings and finish the wizard
1️⃣1️⃣ Open Task Scheduler and confirm the task was created
1️⃣2️⃣ Check that it is configured the way you want
1️⃣3️⃣ Right-click the task and select Run to test that the backup job starts before the scheduled time ✅🚀
